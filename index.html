Introducción

Este sitio web busca ofrecer una visión clara y práctica del rol de QA, con un énfasis especial en el desarrollo utilizando el lenguaje Java. Presentamos conceptos clave, habilidades necesarias y criterios de validación de código.

Objetivos del Proyecto:

  • - Comprender el rol de QA y sus habilidades esenciales.
  • - Explorar Java y su relevancia en aplicaciones web.
  • - Establecer criterios para la validación de código fuente.
  • - Crear un sitio HTML estático con Bootstrap y diseño corporativo.
